Method Circle
Draws a circle starting from the current x and y position of the pen.
The circle is drawn around the current pen position, with the x and y co-ordinate at the centre.
Syntax
C#
    public void Circle(int radius, bool filled)
Parameters
radius
Type: int
Defines the radius of the circle: the distance between the centre (current pen psoition) and the outer edge of the circle.
filled
Type: boolean
States wether or not the circle will be filled with colour or just provide an outline.
Exceptions
Exception Condition
CanvasException The radius provided is less than zero
Examples
canvas.Circle(50, false);

Method DrawTo
Draws a line from the current co-ordinate position of the pen to the specified co-ordinates.
End co-ordinates are passed in as parameters.
Draws with the current pen's state, the line can vary in colour, thickness and opacity.
Syntax
C#
    public void DrawTo(int toX, int toY)
Parameters
toX
Defines the x co-ordinate that the line should draw to.
toY
Defines the x co-ordinate that the line should draw to.
Exceptions
Exception Condition
CanvasException The x or y co-ordinates are each either less than zero, or out of bounds of the canvas.
Examples
canvas.DrawTo(40,50);
